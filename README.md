# 🌦️ Weather App

A weather forecast application that shows real-time weather and a 5-day forecast using the OpenWeatherMap API. Built with vanilla JavaScript (frontend) and Express (backend) for API key protection.

---

## 🔧 Features

- 🌍 Detects user location via geolocation
- 🕒 Displays current time and date
- 🌤️ Shows current weather with icon, temperature, humidity, pressure, wind speed
- 📅 5-day forecast with daily icons and min/max temperature
- 🔐 Backend proxy to protect API key

---

## 🛠️ Technologies

- HTML, CSS, JavaScript (Frontend)
- Node.js, Express (Backend)
- OpenWeatherMap API
- Moment.js (Date formatting)
